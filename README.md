# Sobre
- Aplicação de gestão de compras para setor varejista;
- Sugestão de pedido baseado no estoque e giro de produtos;
- Relatório em xls;

## Modulos:
- Cadastro de Usuario;
- Cadastro de Empresa;
- Cotação de Produtos;
- Pedido de compra;
- Compra x Venda;
